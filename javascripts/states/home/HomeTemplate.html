angular-mapboxgl-directive

angular-mapboxgl-directive is a directive for AngularJS 1.x that allows you to create Mapbox GL maps easily, on your AngularJS applications. With this directive, you can get the full power of Mapbox GL in your AngularJS application.

$ npm install angular-mapboxgl-directive --save

angular-mapboxgl-directive

This directive for angular 1.x, allows you to create Mapbox GL maps easily on your AngularJS applications.

This project has been inspired by the angular-leaflet-directive project developed by tombatossals.

Get Started

You can install the directive using NPM or Bower:

NPM
$ npm install angular-mapboxgl-directive --save
Bower
$ bower install angular-mapboxgl-directive --save
